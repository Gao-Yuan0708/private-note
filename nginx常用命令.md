#### 安装Command Line tools
`xcode-select --install`

#### 安装brew命令、
`ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

#### 安装nginx
`brew install nginx`

#### 启动nginx
`sudo nginx`

#### 关闭nginx命令
`sudo nginx -s stop`

####  重新启动nginx
`sudo nginx -s reload`

#### 打开nginx目录
`open /usr/local/etc/nginx/`
